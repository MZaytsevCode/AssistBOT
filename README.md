#### Bot Commands:
 -  ! ping is your ping

 -  ! 8ball "question" - Ball of predictions

 -  ! clear "Qty" - Clear chat;

 -  ! ban "@ nickname" - Block user;

 -  ! unban "nickname # personal tag" - Unblock users;

 -  ! wb "color white / black" - The game is white / black;

 -  ! com - Bot commands;

 -  ! users - Chat users;

 -  ! spam + count - Spam from the bot;

 -  ! users - Chat users;
